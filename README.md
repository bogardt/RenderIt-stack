# RenderIt-stack
Stack docker pour un projet d'ecole javascript full stack (nodeJS/reactJS) 

1) Build the stack
```bash
docker-compose build --no-cache
```

2) Launch it
```bash
docker-compose up
```

# Utils

#### Down the stack
```bash
docker-compose down
```

#### Delete databases
```bash
sudo rm -rf mongo_data
```
# Webapi
Api is on port 4000

# Websocket
Tchat running with websocket on port 4001

# Website
Webapp is on port 3000
